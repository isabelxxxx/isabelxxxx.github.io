01 逾期资产流转路径
内部自催阶段
逾期 M1 – M3
🏦 债权所有权：金融机构持有，关系完整

借款人逾期后1–3个月，由金融机构内部催收部门主导处置。此阶段债权无任何转移，合规成本最低，也是回收率最高的窗口期——实务中通常能回收逾期账户中60%–70%的余额。催收方式以电话、短信、信函为主，配合内部风控模型进行账户分层和优先级排序。

内部催收团队

委外催收阶段
逾期 M4 – M6
📋 债权所有权：仍归金融机构 — 委托代理关系，债权不转移

金融机构将催收业务委托给外部机构，但债权所有权不发生任何转让。外部机构以受托代理人身份操作，按实际回收金额的一定比例收取佣金（通常15%–40%）。需特别注意：委外机构的违规行为，法律责任仍由金融机构承担，因此须对委外方进行持续的资质审查与行为管控。

逾期 M7 – M12
⚖️ 债权所有权：路径分叉 — 转让则转移，诉讼则不变

此阶段金融机构面临两条路径选择，所有权归属因路径不同而分叉：

路径 A · 批量债权转让
将资产以面值20%–50%的折扣价打包出售给AMC或其他受让方。债权所有权完整转移，金融机构实现风险出表，AMC成为新债权人。
路径 B · 司法追偿
向法院提起诉讼或申请仲裁，获得生效判决后申请强制执行、财产查封或拍卖。债权所有权保持不变，金融机构作为债权人继续追索。

📑 债权所有权：核销≠放弃债权，账外持有；AMC持有的资产归AMC

金融机构将长期无法回收的资产计提损失、移出资产负债表（"出表"），但债权在法律上依然存续，不因核销而消灭。核销后资产可以继续委外催收，也可以更低折扣再次转让。与此同时，前阶段已转让给AMC的资产，在AMC内部进入重组处置流程。

03 关键实务要点
KEY POINT 01
委外不等于免责
金融机构将催收业务委托外部机构后，债权所有权不发生转移，委外机构的违规催收行为（如骚扰、威胁）所产生的法律责任依然由金融机构承担，须对委外方进行持续的准入审查和过程管控。
KEY POINT 02
核销≠放弃债权
账面核销是会计处理，金融机构将资产移出资产负债表、计提损失，但债权在法律上依然存续。在诉讼时效（通常3年）届满前，该债权仍可再委托催收、低价出售或通过法律手段追索。
KEY POINT 03
折扣率的定价博弈
批量转让时，AMC收购价通常为面值的20%–50%（消费信贷类资产通常更低）。收购价偏低则金融机构账面损失扩大，收购价偏高则AMC回报空间收窄——双方博弈的核心是对资产未来回收率的预判差异。
KEY POINT 04
数据合规新约束
《个人信息保护法》实施后，债权转让须同步解决借款人个人信息的授权传递问题。境外AMC或外资基金入场还需满足数据出境安全评估要求，增加了交易的合规成本与时间成本。
KEY POINT 05
不良资产转让须公开竞价
金融机构将不良资产包转让给AMC，通常须通过公开招标或竞价方式进行，不得私下点对点交易，以防止利益输送和国有资产流失。银行业信贷资产登记流转中心（银登中心）是法定挂牌平台，资产包须在此公示后方可转让。
KEY POINT 06
个人贷款批量转让2021年才开闸
2021年前，监管明确禁止个人贷款不良资产批量转让给AMC，只有对公贷款（企业贷款）可以打包出售。2021年原银保监会发布试点政策，个人消费贷、信用卡不良资产方可通过银登中心批量转让，目前仍处于试点扩容阶段。

监管框架
